# Buyer
Buyer is e-commerce web application powered by Node.js to concretize my theoretical information and improve myself. It has not been completed yet and I am still working on it.

# Used Tecnologies
- Javascript
- Node.js
- HTML, CSS, Handlebars
- MongoDB
